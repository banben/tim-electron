# tim-electron 用electron开发的仿腾讯TIM界面

目前只做了登录界面
以后会更新主界面

###### 更多信息请访问 [cnygf.net](http://www.cnygf.net).

## 快速开始
#### 下载和编译
    git clone https://github.com/cn-ygf/tim-electron.git
    安装electron https://github.com/electron/electron/releases/
#### 运行
    把electron加入到环境变量
    cd tim-electron
    electron .
